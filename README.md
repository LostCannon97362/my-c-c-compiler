# Simple C/C++ Compiler for Visual Studio Code

This extension adds a command to compile the currently open C or C++ file using gcc or g++. Requires Visual Studio Code, and gcc/g++ installed on your system.

## Features
- Compile the active C or C++ file with a single command
- Output appears in the VS Code terminal

## Usage
1. Open a C or C++ file in VS Code.
2. Open the Command Palette (F1 or Ctrl+Shift+P) and type `Compile C/C++`.
3. The output/executable will be created in the same directory as your source file.

## Requirements
- Visual Studio Code
- gcc or g++ installed (in PATH)

## Install
- Clone/download this repo
- Run `npm install` then `npx tsc`
- Press F5 to open an Extension Development Host window
